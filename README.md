# ITCamp22 Mini Project

## Concept

เกม Dungeon ที่ผู้เล่นจะต้องปราบ Enemy และนำกระดิ่งนำโชคทั้ง 4 ไปใส่ให้ครบให้ได้

## Gameplay Condition

ถ้าตายครบ 5 ครั้ง -> แพ้
ถ้าเก็บ Artifact ครบ 4 อัน -> ชนะ

## Reference

- [Archemara (Sabotage)](https://www.roblox.com/games/13100646976/ARCHEMARA)

## Mechanic

- Pickup / Drop
- Combat
- Critical Damage
- Zoom Camera
- Stamina

## Technical System

- Multiple UI Handler (Blueprint Component)
- Game Instance (Level Switching)
- Blueprint Inheritance (Enemy, Weapon)
